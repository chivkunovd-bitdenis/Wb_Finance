# Zurich VPS (proxy/Tor)

## Адрес

- **Host/IP**: `194.87.96.144`

## Для чего используется

- Точка входа по SSH для проброса SOCKS (Tor) на `localhost:9050`, чтобы запускать отдельное окно Chrome с трафиком через Tor.

## Заметки

- SSH user обычно `root` (если менялся — обновить здесь).
- Tor SOCKS ожидается на VPS: `127.0.0.1:9050`.

## WB Finance на этом же VPS

- **Каталог проекта на сервере**: `/root/wb-finance` (не `/opt/wb-finance`).
- Деплой и обновления — **только из Git**: в каталоге **должна** быть папка `.git`. Если проект когда‑то залили **zip/scp/rsync без `.git`**, команда `git pull` **всегда** будет падать с `fatal: not a git repository` — это не «сломался сервер», а способ выкладки. См. `КАК_ОБНОВИТЬ_СЕРВЕР.md` → раздел про отсутствие `.git`.

# WB Finance Pro — фронтенд

React (Vite) + Bootstrap + Chart.js. Интерфейс повторяет текущий GAS-прототип: логин/регистрация, дашборд, артикулы, себестоимость, воронка.

## Запуск в режиме разработки

1. Убедись, что бэкенд уже запущен (Docker: `docker compose up` в корне проекта, API на порту 8000).
2. В папке `frontend` выполни:
   ```bash
   npm install
   npm run dev
   ```
3. Открой в браузере: http://localhost:5173

По умолчанию фронт ходит на API по адресу `http://localhost:8000`. Если API на другом хосте/порту, создай файл `frontend/.env` и укажи:
```
VITE_API_URL=http://твой-хост:порт
```

## Сборка для продакшена

```bash
npm run build
```
Собранные файлы появятся в папке `dist/`. Их можно отдавать любым веб-сервером (Nginx, Apache и т.д.) или положить в статику бэкенда.
